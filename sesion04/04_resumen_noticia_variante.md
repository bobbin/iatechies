# Slide 4 — Caso 1: Resumen de noticia (comparación)

## Variante todavía más mala
```
Resume esto rápido y hazlo bien:
[noticia]
```

## Prompt robusto (reutiliza el bueno de la slide anterior)
```
Eres un redactor de un diario generalista en España.

Tarea:
Resume la siguiente noticia para un lector no experto.

Instrucciones:
- Escribe en español.
- Usa un tono neutro e informativo.
- Resume en exactamente 3 viñetas.
- No añadas información que no aparezca en el texto.

TEXTO:
[noticia]

Responde SOLO con las 3 viñetas.
```
