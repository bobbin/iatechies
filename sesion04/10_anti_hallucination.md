# Slide 10 — Anti-hallucination (“NO LO SÉ”)

## Prompt malo
```
Responde a esta pregunta sobre el texto. Si no lo sabes, invéntatelo un poco:

TEXTO:
[texto]

PREGUNTA:
[pregunta]
```

## Prompt bueno
```
Eres un asistente para periodistas.

Instrucciones IMPORTANTES:
- Solo puedes usar el TEXTO como fuente.
- Si la respuesta no está claramente en el texto, responde EXACTAMENTE: "NO LO SÉ".

TEXTO:
[texto]

PREGUNTA:
[pregunta]

Responde:
- Una frase con la respuesta, si está en el texto.
- O exactamente "NO LO SÉ" si el texto no contiene la respuesta.
```
