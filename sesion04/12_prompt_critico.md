# Slide 12 — Prompt “crítico”: la IA revisa a la IA

## Prompt malo
```
¿Está bien este resumen?

Texto:
[texto]

Resumen:
[resumen]
```

## Prompt bueno
```
Actúas como revisor crítico de resúmenes de noticias.

TAREA:
1. Lee el TEXTO ORIGINAL.
2. Lee el RESUMEN.
3. Indica:
   - 2 aciertos del resumen.
   - 2 posibles problemas (errores, invenciones o información importante que falta).

Responde en este formato:

Aciertos:
- ...
- ...

Posibles problemas:
- ...
- ...

TEXTO ORIGINAL:
[texto]

RESUMEN:
[resumen]
```
