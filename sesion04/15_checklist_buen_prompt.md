# Slide 15 — Checklist de buen prompt

## Prompt sin checklist
```
Explica esto:

[texto]
```

## Prompt usando la checklist
```
Eres un divulgador científico.

Tarea:
Explicar el contenido del siguiente texto a un público general.

Instrucciones:
- Escribe en español.
- Máximo 4 frases.
- Tono claro y cercano, sin jerga técnica.
- No inventes información que no esté en el texto.

Formato de salida:
Un único párrafo de 3–4 frases.

Texto:
[texto]
```
